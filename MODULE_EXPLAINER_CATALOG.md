# Module Explainer Catalog

For every major module, Ultimate Church OS answers: **What is this?** **Why use it?** **When?** **Who?** **Examples.**

Browse in the app: **Academy → Module guide** (opens modules with **Open module**).

---

## Home

| | |
|--|--|
| **What** | Church overview — people, giving, events, and what needs attention this week. |
| **Why** | See church health at a glance without opening every module. |
| **When** | Start of the week, Monday staff meeting, or before Sunday. |
| **Who** | Pastors, church administrators, leadership |
| **Examples** | Check volunteer gaps before Sunday; review giving pace vs budget |

---

## Members

| | |
|--|--|
| **What** | People directory — visitors, members, leaders, contact details. |
| **Why** | One place to find anyone and their journey stage. |
| **When** | Follow-up calls, visitor assimilation, updating contact info. |
| **Who** | Pastors, administrators, small group leaders |
| **Examples** | Find a family before a pastoral visit; move Visitor → Member |

---

## Families

| | |
|--|--|
| **What** | Household groupings linked to member profiles. |
| **Why** | Care and communication often happen at family level. |
| **When** | Registering couples/children or updating household address. |
| **Who** | Administrators, pastors |
| **Examples** | Open the Nair family before a home visit |

---

## Volunteers

| | |
|--|--|
| **What** | Who serves where — event roles, ministries, Sunday teams. |
| **Why** | Coordinate teams without stale spreadsheets. |
| **When** | Planning Sunday, event rosters, finding substitutes. |
| **Who** | Volunteer coordinators, worship leaders, administrators |
| **Examples** | Assign greeters for 9 AM; see youth night confirmations |

---

## HR & Staff

| | |
|--|--|
| **What** | Church employees — job records, leave, payroll setup, documents. |
| **Why** | Separate paid staff workflows from congregation members. |
| **When** | Hiring, leave approval, month-end payroll with finance. |
| **Who** | HR managers, administrators, finance (payroll) |
| **Examples** | Approve associate pastor leave; review worship leader profile |

---

## Pastoral Care

| | |
|--|--|
| **What** | Care cases, follow-ups, pastoral notes. |
| **Why** | Nothing falls through when multiple pastors share load. |
| **When** | Counseling requests, hospital visits, benevolence. |
| **Who** | Pastors, care team, small group leaders (limited) |
| **Examples** | Track hospital follow-up; assign case to associate |

## Small Groups

| | |
|--|--|
| **What** | Home groups — rosters and meeting rhythm. |
| **Why** | Mid-week community is where much discipleship happens. |
| **When** | Group season launch, attendance, leader changes. |
| **Who** | Small group leaders, discipleship pastors |
| **Examples** | Anna Nagar home group roster; Thursday attendance |

## Events

| | |
|--|--|
| **What** | Calendar — services, conferences, classes, outreach. |
| **Why** | Events drive volunteers, attendance, and comms in one record. |
| **When** | VBS, seminars, weekly worship services. |
| **Who** | Administrators, ministry leaders, worship team |
| **Examples** | Create VBS; open Sunday 9 AM workspace |

## Sunday Service

| | |
|--|--|
| **What** | Live-service control center for worship in progress. |
| **Why** | Run flow, monitor teams, attendance, and issues on one screen. |
| **When** | **During** Sunday worship — not week-ahead planning. |
| **Who** | Pastors, worship leaders, volunteer coordinators |
| **Examples** | Advance worship segments; see media and team alerts |

*Requires event `type: Service` — see Grace demo `ev-sunday`.*
