# HR Manager — Quick Start

## First day (about 3 hours)

1. Sign in — lands on **HR & Staff**.
2. Link staff **members** to employment profiles.
3. Configure **leave balances** for active staff.
4. Review open **leave requests** and approval flow.
5. Coordinate **payroll** with Finance (Vendors & Payroll).

## First week

| Focus | Action |
|-------|--------|
| Onboarding | HR → onboarding tasks for new hires |
| Leave | Approve/deny requests promptly |
| Documents | Staff documents on member profile |

## Key workflows

- **New hire:** Members → workforce class → HR profile → leave balances.
- **Leave:** Staff requests → HR approves → balance updates.
- **Payroll:** Configure structures in UI before first run.

## Gaps to plan around

- Offboarding is manual today (status + end date on employment profile).
