# Finance Manager — Quick Start

## First day (about 4 hours)

1. Sign in — you land on **Finance** → vouchers.
2. **Settings** → confirm chart of accounts and financial year.
3. **Giving** → record a test gift and confirm receipt.
4. **Finance** → create draft voucher → approve → post.
5. **Finance** → reconciliation tab — understand CSV import.

## First week

| Day | Task |
|-----|------|
| Mon–Fri | Record gifts, clear draft vouchers |
| Fri | Review bank reconciliation |
| Month-end | Financial year reports, audit trail |

## Key workflows

- **Gift:** Giving → Record gift → receipt.
- **Expense:** Finance → New voucher → balanced lines → approve → post.
- **Correction:** Posted voucher → reverse.

## Tips

- Budget detail lives in **Budgets** module.
- Use **Guide** → Finance track for click-through steps.
